# 나만의 포트폴리오 웹사이트

이 프로젝트는 개인 포트폴리오를 위한 반응형 웹사이트입니다. HTML, CSS, JavaScript를 사용하여 제작되었으며, 모바일에서도 잘 작동합니다.

## 기능

- 반응형 디자인 (모바일, 태블릿, 데스크톱 지원)
- 부드러운 스크롤 네비게이션
- 스크롤 시 페이드인 애니메이션
- 연락처 링크 및 소셜 미디어 링크
- 프로젝트 카드 레이아웃

## 사용 방법

1. 이 저장소를 클론하거나 다운로드하세요.
2. `index.html` 파일을 웹 브라우저에서 엽니다.

## 커스터마이징

- `index.html`: 콘텐츠 수정
- `styles.css`: 스타일 수정
- `script.js`: 인터랙티브 기능 수정
